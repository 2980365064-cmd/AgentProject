# AgentProject - 智能医院管理系统
## 🏥 医院管理 + AI服务一体化平台

本项目是一套**前后端分离 + AI智能服务**的综合医院管理系统，
包含**管理员后台**、**患者用户端**、**Python AI服务模块**，实现医院数字化、智能化管理。

---

## 📌 项目架构
项目由三大模块组成：

### 1. medical-ai-gateway（Python AI模块）
- 语言：Python
- 功能：提供AI智能服务、数据分析、智能预测等能力
- 为医院系统提供智能化扩展支持

### 2. hospital（Java SpringBoot 后端）
- 语言：Java
- 框架：SpringBoot
- 功能：医院管理系统核心接口服务
- 支持**管理员 / 患者**双角色权限体系

### 3. Frontend（React 前端）
- 语言：JavaScript / React
- 功能：后台管理界面 + 患者用户界面
- 响应式布局，美观易用

---

## 🎯 系统功能介绍

### 🔹 管理员端（后台管理）
管理员登录后可进行全系统管理：
- 医生信息管理（增删改查）
- 患者入院/出院管理
- 工作台实时查看
- 查看当前在岗医生
- 查看在院患者列表
- 查看待出院患者
- 系统数据统计与报表
- 权限管理

### 🔹 患者端（用户中心）
患者注册/登录后可使用：
- 在线挂号预约
- 查看个人病历
- 查看住院记录
- 查看检查报告
- 个人信息管理
