# 🏥 Hospital Management System – Backend API

This repository contains the backend RESTful API for the **Hospital Management System**, a scalable healthcare platform designed to efficiently manage hospital operations.

Built using **Java**, **Spring Boot**, and **MySQL**, this system powers full CRUD operations for doctors, wards, patients, appointments, and treatments while ensuring secure and structured healthcare data management.

## 🚀 Features

- **Doctor Management**
  - Register, update, delete, and view doctor details
  - Manage specialties and availability

- **Patient Management**
  - Maintain patient medical records
  - Track patient history and treatments

- **Ward Management**
  - Organize hospital wards
  - Manage ward capacity and assignments

- **Appointment Scheduling**
  - Book and manage appointments
  - Link patients with doctors efficiently

- **Treatment Management**
  - Record treatments and prescriptions
  - Maintain structured healthcare data

- **RESTful Architecture**
  - Clean API design following REST principles

- **Full CRUD Operations**
  - Create, Read, Update, and Delete functionality across all modules

---

## 🧰 Tech Stack

- **Language**: Java  
- **Framework**: Spring Boot  
- **Database**: MySQL  
- **ORM**: Spring Data JPA / Hibernate  
- **Build Tool**: Maven  
- **API Design**: RESTful API  
- **Architecture**: Layered Architecture (Controller → Service → Repository)

---

## 🏥 System Modules

- Doctors
- Patients
- Wards
- Appointments
- Treatments

Each module is designed independently to support scalability and maintainability.
