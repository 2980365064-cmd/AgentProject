# 🏥 Hospital Management System – Frontend

This repository contains the frontend Single Page Application (SPA) for the **Hospital Management System**.  

Built using **React**, this responsive web application provides an intuitive and mobile-friendly interface for managing hospital operations including doctors, wards, patients, appointments, and treatments.

The application communicates with any RESTful backend API to perform full CRUD operations and streamline healthcare management workflows.

## 🚀 Features

- Responsive Single Page Application (SPA)
- Mobile-friendly modern UI
- Full CRUD Operations
- RESTful API Integration
- Dynamic Routing using React Router
- Real-time data fetching using Fetch API
- Clean and intuitive hospital management dashboard

---

## 🧰 Tech Stack

- **Framework**: React.js
- **Routing**: React Router
- **HTTP Client**: Fetch API
- **Styling**: CSS / Bootstrap / Custom Styles
- **Architecture**: Component-Based Architecture
- **API Communication**: RESTful API Integration

---

## 🏥 System Modules

- Doctors Management
- Patients Management
- Wards Management
- Appointment Scheduling
- Treatment Records

Each module allows users to create, view, update, and delete records seamlessly.
