▌1. 常见症状→初步诊断 + 建议（10 条，直接用）
1）急性发热（37.5–39℃，＜7 天）
可能原因：普通感冒、急性咽炎、流感、急性支气管炎、尿路感染（女性）。
家庭护理：多喝水、休息、物理降温（温水擦身）；体温≥38.5℃可服对乙酰氨基酚 / 布洛芬（无过敏）。
就医指征：持续＞3 天不退；高热≥39.5℃；伴呼吸困难、胸痛、意识模糊、剧烈头痛、呕吐、皮疹、小便刺痛。
科室：发热门诊 / 全科。
2）咳嗽（急性＜3 周）
可能原因：上呼吸道感染、急性支气管炎、过敏、冷空气刺激。
家庭护理：多喝水、蜂蜜水（≥1 岁）、加湿空气；避免烟酒 / 辛辣。
就医指征：咳黄脓痰 / 血痰；胸痛、气短；发热＞38.5℃＞3 天；咳嗽＞2 周。
科室：呼吸内科 / 全科。
3）头痛（紧张性 / 偏头痛）
可能原因：熬夜、压力、颈椎病、偏头痛、感冒、高血压。
家庭护理：休息、避光安静环境、热敷颈肩；避免咖啡 / 浓茶。
就医指征：突发剧烈 “爆炸样” 头痛；伴高烧、颈硬、呕吐、肢体无力、言语不清；外伤后头痛。
科室：神经内科 / 全科。
4）急性腹痛（＜24 小时）
可能原因：胃肠炎、消化不良、便秘、痛经、急性阑尾炎、胆囊炎。
家庭护理：暂禁食或清淡流食；热敷（非剧痛 / 高烧）；避免止痛药（掩盖病情）。
就医指征：剧痛难忍、持续加重；高烧≥38.5℃；呕吐 / 停止排便排气；右上腹 / 右下腹固定压痛；孕妇 / 老人腹痛。
科室：急诊科 / 消化内科。
5）鼻塞流涕（急性）
可能原因：普通感冒、过敏性鼻炎、急性鼻窦炎。
家庭护理：生理盐水洗鼻、多喝水、加湿空气；过敏者远离花粉 / 尘螨。
就医指征：流脓涕伴面部压痛＞10 天；持续鼻塞影响睡眠；反复鼻出血。
科室：耳鼻喉科 / 全科。
6）胸痛（非心脏急症）
可能原因：肋间神经痛、肌肉拉伤、胃食管反流、焦虑。
家庭护理：休息、避免剧烈运动；反流者睡前 3 小时不进食、垫高床头。
⚠️ 立即就医（心梗 / 肺栓塞）：压榨样剧痛、放射至左肩 / 下颌；伴大汗、呼吸困难、濒死感；突发剧烈胸痛 + 咯血 / 气促。
科室：急诊科 / 心内科。
7）头晕 / 眩晕
可能原因：低血糖、贫血、颈椎病、耳石症、高血压 / 低血压、睡眠不足。
家庭护理：缓慢起身、避免快速转头；低血糖喝糖水；保证睡眠。
就医指征：眩晕持续＞1 小时；伴剧烈头痛、呕吐、肢体麻木无力、言语不清、视物重影。
科室：神经内科 / 全科。
8）腹泻（急性＜2 周）
可能原因：急性胃肠炎、饮食不洁、病毒感染、受凉。
家庭护理：口服补液盐（防脱水）、清淡饮食（粥 / 面）；避免油腻 / 生冷 / 牛奶。
就医指征：腹泻＞6 次 / 天、伴严重脱水（口干、尿少、头晕）；便血 / 黏液便；高烧＞38.5℃＞2 天；孕妇 / 老人 / 儿童。
科室：消化内科 / 急诊科。
9）皮肤瘙痒 / 皮疹
可能原因：湿疹、荨麻疹、接触性皮炎、蚊虫叮咬、过敏。
家庭护理：避免搔抓、温水洗澡、保湿；远离可疑过敏原（新衣物 / 化妆品）。
就医指征：全身大面积皮疹、瘙痒难忍影响睡眠；伴发热、水疱、渗液、破溃；用药后加重。
科室：皮肤科。
10）失眠（短期＜3 个月）
可能原因：压力、焦虑、熬夜、咖啡 / 浓茶、环境嘈杂。
家庭护理：固定作息、睡前 1 小时远离电子屏幕；避免咖啡 / 浓茶 / 酒精；卧室黑暗安静。
就医指征：失眠＞3 个月；伴严重焦虑 / 抑郁、头痛、记忆力下降；影响日间工作生活。
科室：神经内科 / 心理科。
▌2. 常见慢性病（患者咨询高频）
高血压（成人）
诊断标准：非同日 3 次，收缩压≥140mmHg 和 / 或舒张压≥90mmHg。
生活建议：低盐（＜5g / 天）、低脂饮食；规律运动（快走 / 慢跑，每周≥150 分钟）；戒烟限酒；控制体重；避免熬夜 / 情绪激动。
用药提示：遵医嘱长期服药，不可自行停药 / 减量；定期监测血压（早 / 晚各 1 次）。
就医指征：血压≥180/120mmHg（急症）；伴头痛、呕吐、胸痛、肢体无力、视物模糊。
2 型糖尿病（成人）
典型症状：多饮、多食、多尿、体重减轻；部分患者无症状。
生活建议：控制总热量，低糖低脂高纤维饮食；规律运动（餐后快走 30 分钟）；控制体重；戒烟限酒。
用药提示：遵医嘱服药 / 注射胰岛素；定期监测血糖（空腹 + 餐后 2 小时）。
就医指征：血糖持续＞13.9mmol/L；伴口渴严重、呕吐、腹痛、意识模糊（酮症酸中毒）；反复感染、伤口不愈。
▌3. 用药安全（通用，禁止推荐具体药名）
任何药物均需排除过敏，遵医嘱或说明书服用。
避免滥用抗生素（仅细菌感染有效，病毒感冒无效）。
儿童 / 孕妇 / 哺乳期用药需专科医生指导。
服药期间避免酒精、浓茶、咖啡（影响药效或增加副作用）
